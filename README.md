# 航大CBT 過去問演習アプリ

航空大学校第一次試験（総合 Part II）の過去問演習Webアプリです。

## 収録年度
- 令和6年度（2024年度）全25問

## 技術スタック
- React 18
- Vite
- Vercel（ホスティング）

## ローカル開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## 出典
問題・選択肢は航空大学校公式HP公開の過去問PDFより使用。
解説・UI・アプリ機能は独自制作。

https://www.kouku-dai.ac.jp/nyuusi/kakomonannnai.htm
